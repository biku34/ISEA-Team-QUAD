# ============================================================================
# EXTERNAL SYSTEM TOOLS REQUIRED (not Python packages)
# ============================================================================
# These must be installed on the system separately using apt/yum/brew
#
# 1. The Sleuth Kit (TSK) - Core forensic analysis toolkit
#    - Installation: sudo apt-get install sleuthkit
#    - Provides: mmls, fls, icat, istat, fsstat, etc.
#    - Version: 4.11.0 or higher recommended
#    - Purpose: Partition detection, file system analysis, file recovery
#
# 2. libewf (Expert Witness Format Tools)
#    - Installation: sudo apt-get install libewf-tools ewf-tools
#    - Provides: ewfmount, ewfinfo, ewfacquire
#    - Version: 20140807 or higher
#    - Purpose: Mount E01/EWF forensic images
#
# 3. Scalpel - File carving tool
#    - Installation: sudo apt-get install scalpel
#    - Config: /etc/scalpel/scalpel.conf
#    - Version: 2.0 or higher
#    - Purpose: Recover deleted files from unallocated space
#
# 4. Optional but Recommended:
#    - afflib-tools: Advanced Forensic Format support
#      - Installation: sudo apt-get install afflib-tools
#    - bulk_extractor: Advanced data extraction
#      - Installation: sudo apt-get install bulk-extractor
#
# Verification Commands:
#    mmls -V              # Check Sleuth Kit version
#    ewfinfo -V           # Check libewf version
#    scalpel -V           # Check Scalpel version
#
# ============================================================================

# ============================================================================
# INSTALLATION INSTRUCTIONS
# ============================================================================
#
# 1. Install Python dependencies:
#    pip install -r requirements.txt
#
# 2. Install system forensic tools (Ubuntu/Debian):
#    sudo apt-get update
#    sudo apt-get install -y sleuthkit libewf-tools ewf-tools scalpel
#
# 3. For RHEL/CentOS/Fedora:
#    sudo yum install -y sleuthkit libewf ewf-tools scalpel
#
# 4. For macOS (using Homebrew):
#    brew install sleuthkit libewf scalpel
#
# 5. Create necessary directories:
#    mkdir -p forensics/{evidence,mounts,output,recovered,carved}
#
# 6. Set up configuration:
#    cp config/config.example.yaml config/config.yaml
#    # Edit config.yaml with your settings
#
# 7. Initialize database:
#    alembic upgrade head
#
# 8. Run the application:
#    uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
#
# ============================================================================
